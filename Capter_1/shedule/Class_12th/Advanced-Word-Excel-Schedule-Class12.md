# Advanced MS Word & MS Excel — Day-wise Schedule (Class 12)

**For students who already know:** Basic Word formatting, Basic Excel, VLOOKUP, INDEX-MATCH
**Duration:** 6 Weeks
**Goal:** Office-level / exam-level advanced skills

---

## WEEK 1: Excel — Advanced Lookup & Logical Functions

**Day 1** — Revision: VLOOKUP, INDEX-MATCH ki limitations samjhao (kyu INDEX-MATCH better hai VLOOKUP se)
**Day 2** — HLOOKUP + XLOOKUP (naya function, agar Excel version support kare)
**Day 3** — Nested IF (multiple conditions ek saath)
**Day 4** — IF + AND / IF + OR combination
**Day 5** — IFERROR, IFNA (errors ko handle karna) + practice sheet

## WEEK 2: Excel — Data Analysis Tools

**Day 6** — Conditional Formatting (color scales, data bars, icon sets, custom rules)
**Day 7** — Data Validation (dropdown list banana, input restriction)
**Day 8** — Text Functions: LEFT, RIGHT, MID, CONCATENATE/TEXTJOIN, TRIM
**Day 9** — Date & Time Functions: TODAY, NOW, DATEDIF, YEAR/MONTH/DAY
**Day 10** — SUMIF, SUMIFS, COUNTIF, COUNTIFS (conditional calculations) + practice

## WEEK 3: Excel — Pivot Tables & Charts

**Day 11** — Pivot Table kya hota hai, kaise banate hain (basic example: sales data)
**Day 12** — Pivot Table: Filters, Rows, Columns, Values arrange karna
**Day 13** — Pivot Chart banana
**Day 14** — Advanced Charts: Combo chart, Line + Bar together
**Day 15** — Sparklines + Freeze Panes + Split view (large data handle karna)

## WEEK 4: Excel — Automation Basics

**Day 16** — Named Ranges ka use
**Day 17** — Data from multiple sheets link karna (=Sheet1!A1 type formulas)
**Day 18** — Protect Sheet/Workbook (password lagana, cells lock karna)
**Day 19** — Introduction to Macros (Record Macro — bina coding ke automation)
**Day 20** — Full Practical: Ek mini "Student Result Analyzer" banwao (VLOOKUP + IF + Pivot combine karke)

---

## WEEK 5: Word — Advanced Features

**Day 21** — Styles aur Formatting (Heading 1, 2, 3 use karna professional documents ke liye)
**Day 22** — Table of Contents (automatic, Styles se link hoke banta hai)
**Day 23** — Mail Merge (ek letter multiple logo ko bhejne ke liye — bahut useful office skill)
**Day 24** — Track Changes aur Comments (group projects/editing ke liye)
**Day 25** — SmartArt, Watermark, Section Breaks (different page orientation same doc me)

## WEEK 6: Integration + Real Projects

**Day 26** — Excel data ko Word me link/embed karna (report banane ke liye)
**Day 27** — Resume/CV banwao Word me (real use case, Styles + Tables use karke)
**Day 28** — Excel me ek Budget Tracker ya Attendance Sheet banwao (formulas + conditional formatting)
**Day 29** — Doubt clearing + revision
**Day 30** — Final Project Day: Har student ek chhota project submit kare — either "School Result Sheet" (Excel) ya "Formal Report" (Word) jisme sab advanced features use ho

---

## Teaching Tips for Advanced Batch

1. **Real-life examples use karo** — sirf formula mat sikhao, batao "ye office me kaha use hota hai" (jaise HR data, sales report, attendance).
2. **Har concept ke baad ek chhota challenge do** — "isse khud solve karo" — advanced students ko spoon-feeding nahi, thoda struggle karne do.
3. **Group/pair projects** karwao Week 6 me — real files pe kaam karne se confidence aata hai.
4. Agar time bache to **Google Sheets** ka bhi thoda intro de sakte ho (formulas same hote hain, sirf sharing/collaboration alag hai) — aajkal companies me common hai.
5. **Shortcuts sikhao** har week (Ctrl+Shift+L for filter, Ctrl+; for date, F4 for repeat, etc.) — ye speed badhata hai aur professional dikhta hai.
