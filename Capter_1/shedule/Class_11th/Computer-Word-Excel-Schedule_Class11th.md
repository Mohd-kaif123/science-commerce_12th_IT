# Computer, MS Word & MS Excel — Day-wise Teaching Schedule (Class 11)

**Total Duration:** 8 Weeks (kam ya zyada adjust kar sakte ho batch ki speed ke hisaab se)
**Rule:** Har din ek naya topic + pichle din ka 5-10 min revision/practice zaroor karwao. Zero-level students ke liye repetition sabse zaroori hai.

---

## WEEK 1: Computer Ki Basic Samajh (Introduction)

**Day 1 — Computer Kya Hai**
- Computer ke parts dikhao: CPU/Tower, Monitor, Keyboard, Mouse, UPS
- Input device vs Output device (simple examples se samjhao)
- Practical: Sabko baari-baari Computer ON karna aur SHUT DOWN karna sikhao (ye sabse important hai, isko skip mat karo)

**Day 2 — Mouse Aur Keyboard Practice**
- Mouse ke click types: Left click, Right click, Double click, Drag
- Keyboard layout: Alphabet keys, Number keys, Enter, Backspace, Space, Shift
- Practical: Paint ya Notepad khol ke free-hand mouse practice + kuch letters type karwao

**Day 3 — Desktop Aur Icons**
- Desktop kya hota hai, Icons kya hote hain
- Start Menu / Taskbar ka use
- Ek program open karna aur close karna (X button)
- Practical: Notepad open/close 5 baar khud se karwao

**Day 4 — Files Aur Folders**
- File kya hoti hai, Folder kya hota hai
- New Folder banana, Rename karna, Delete karna
- Practical: Apne naam ka folder banao, usme 1 Notepad file save karo

**Day 5 — Typing Practice + Revision**
- Basic typing practice (10-15 min, simple sentences)
- Week 1 ka pura revision — sabse pucho "Computer on kaise karte hain", "folder kaise banate hain"

---

## WEEK 2: Windows Basics (Confidence Build Karna)

**Day 6** — Copy, Cut, Paste (files/folders pe) — Ctrl+C, Ctrl+X, Ctrl+V
**Day 7** — Multiple windows kaise switch karte hain (Alt+Tab), minimize/maximize/close
**Day 8** — Internet basics (agar available ho): browser open karna, search karna (optional, agar lab me net hai)
**Day 9** — Saari cheezein revise + chhota practical test (5 questions: folder banao, file rename karo, copy-paste karo)
**Day 10** — Doubt clearing day + jo pichhe reh gaye unko catch-up karwao

> Yaha tak aane ke baad sab students comfortable ho jaayenge PC handle karne me. Ab Word start karo.

---

## WEEK 3: MS Word — Part 1 (Basics)

**Day 11** — Word kya hai, kaise open karte hain, Blank Document kaise banate hain
**Day 12** — Typing + Save karna (Save vs Save As), file ko naam dena
**Day 13** — Text Select karna, Copy-Paste, Cut, Undo/Redo
**Day 14** — Font formatting: Font style, Font size, Bold, Italic, Underline, Font color
**Day 15** — Text Alignment (Left, Center, Right, Justify) + Revision

## WEEK 4: MS Word — Part 2 (Formatting & Design)

**Day 16** — Bullets aur Numbering
**Day 17** — Line spacing, Paragraph spacing
**Day 18** — Insert menu: Table banana (simple table)
**Day 19** — Insert menu: Picture/Image insert karna, resize karna
**Day 20** — Header/Footer, Page Number, Page Orientation (Portrait/Landscape)
**Day 21** — Printing ka concept (Print Preview, Print settings) + Full Practical Test (ek letter/notice type karke format karwao)

---

## WEEK 5: MS Excel — Part 1 (Basics)

**Day 22** — Excel kya hai, use kaha hota hai (real-life examples: marksheet, budget)
**Day 23** — Cell, Row, Column ka concept; Cell address (A1, B2) samjhao
**Day 24** — Data type karna, Save karna
**Day 25** — Cell formatting: Bold, Font size, Color, Borders, Cell color
**Day 26** — Row height/Column width adjust karna + Revision

## WEEK 6: MS Excel — Part 2 (Basic Calculations)

**Day 27** — AutoSum ka use (simple + karna)
**Day 28** — Basic Formula: =A1+B1, =A1-B1, kaise likhte hain, "=" se kaise start hota hai
**Day 29** — SUM formula (=SUM(A1:A5))
**Day 30** — AVERAGE formula
**Day 31** — MAX aur MIN formula + practice sheet (marks ka example lekar)

## WEEK 7: MS Excel — Part 3 (Thoda Advanced)

**Day 32** — COUNT aur COUNTA formula
**Day 33** — IF formula (bahut simple example se: Pass/Fail marks ke hisaab se)
**Day 34** — Sorting data (A-Z, Z-A)
**Day 35** — Filter ka basic use
**Day 36** — Simple Chart/Graph banana (Bar chart)

## WEEK 8: Revision + Final Practical

**Day 37** — Poora Word revision + ek assignment (notice/letter banwao)
**Day 38** — Poora Excel revision + ek assignment (marksheet banwao formulas ke saath)
**Day 39** — Doubt clearing session
**Day 40** — Final Practical Test (Word + Excel dono)

---

## Teaching Tips (Zaroori Baatein)

1. **Har naya topic sirf 15-20 min theory, baaki time practical** — zero level students ke liye "karke dikhana" sabse zyada kaam karta hai.
2. **Har din start me pichle din ka 5 min quick recap** — isse bhoolte nahi.
3. **Ek topic ek din — do topics ek saath mat do**, warna confusion hoga.
4. **Weekly ek chhota practical test** (Day 5, 10, 15... waghera) — ye pata chalta hai kaun peeche reh gaya.
5. Agar lab me computers kam hain, to 2 students ek PC pe baithao — dono ko baari-baari hands-on do.
6. Slow students ke liye extra 10 min alag se do, unhe group me embarrass mat karo.

---

*Note: Ye schedule flexible hai — agar aapki class fast hai to isko 6 weeks me bhi complete kar sakte ho, ya slow hai to 10 weeks tak khinch sakte ho. Important ye hai ki sequence yahi rahe: Computer Basics → Word → Excel.*
